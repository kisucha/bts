# 테니스 클럽 관리 웹

작은 규모 테니스 클럽의 **순위**, **대진**, **경기 결과**, **개인별/상대별 승률**을 관리하는 모바일 친화 웹앱입니다.

## 사용 방법

1. **실행**: `index.html`을 브라우저에서 열거나, 로컬 웹 서버로 해당 폴더를 서빙합니다.
2. **사용자 로드**: 첫 화면에서 **user.xlsx** 파일을 선택해 사용자 목록을 불러옵니다.
3. **달력**: 월별 달력에서 날짜를 클릭하면 해당 날짜의 경기 화면으로 이동합니다.
4. **경기 시작시간**: 달력 화면·날짜 화면에서 당일 경기 시작시간(기본 7:00)을 선택할 수 있습니다.
5. **참여/체크인/체크아웃**: 해당 날짜에서 참여 여부와 참석 시간(7:00, 7:30, 8:00)을 설정합니다. 체크아웃은 체크인 이후만 선택 가능합니다.
6. **대진표 작성**: 참여자 설정 후 **대진표 작성** 버튼을 누르면 5게임 대진이 생성됩니다. (그레이드별 3게임 + 무작위 2게임, 4명 2v2)
7. **경기 결과**: 각 경기 옆에서 승자(팀1/팀2)를 선택하면 점수와 상대별 승패가 자동 반영됩니다.
8. **순위/승률**: 하단 **순위/승률** 탭에서 전체 순위(점수·승률)를 보고, 사용자를 누르면 해당 인원 대비 **상대별 승률**을 볼 수 있습니다.

## user.xlsx 형식

- **첫 행**: 헤더(컬럼명)
- **이름**: 컬럼명 `이름` 또는 `name` (필수)
- **그레이드**: 컬럼명 `그레이드` 또는 `grade` (선택, 없으면 C). 값은 A, B, C 사용 권장.

예:

| 이름   | 그레이드 |
|--------|----------|
| 홍길동 | A        |
| 김철수 | B        |
| 이영희 | C        |

## 데이터 저장

- 모든 데이터(사용자, 일정, 참여, 대진, 결과, 점수, 상대별 승패)는 **브라우저 localStorage**에 저장됩니다.
- 다른 브라우저나 기기에서는 데이터가 공유되지 않습니다.

## 규칙 요약

- **정기**: 월·수·토 등 원하는 날에 달력에서 날짜를 선택해 경기를 등록합니다.
- **게임**: 1일 5게임. 1~3게임은 그레이드 순(A→B→C)으로 4명씩 매칭, 4~5게임은 무작위.
- **참가 가능 시간**: 체크인 7:30 → 2번째 게임부터, 8:00 → 3번째 게임부터 참가.
- **같은 팀**: 같은 날 이미 같은 팀이었던 조합은 이후 게임에서 같은 팀으로 다시 묶이지 않습니다.
- **점수**: 승리 팀 +1점/명, 패배 팀 -1점/명. 상대별 승패와 승률이 누적됩니다.
